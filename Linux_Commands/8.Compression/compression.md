# Compression Commands

- `tar -cvf archive.tar files` — create a tar archive from files
- `tar -xvf archive.tar` — extract a tar archive
- `tar -czvf archive.tar.gz files` — create a compressed gzip tar archive
- `tar -xzvf archive.tar.gz` — extract a gzip tar archive
- `gzip file` — compress a file using gzip (replaces original file)
- `gunzip file.gz` — decompress a gzip file
- `zip archive.zip files` — create a zip archive
- `unzip archive.zip` — extract a zip archive
