# Process Management Commands

- `ps aux` — display all running processes
- `top` — display real-time process and system information
- `htop` — interactive process viewer (if installed)
- `kill PID` — terminate a process by its process ID
- `killall name` — terminate all processes with the specified name
- `jobs` — list background jobs in the current shell
- `bg` / `fg` — resume a stopped job in the background or foreground
