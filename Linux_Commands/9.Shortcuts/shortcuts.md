# Important Linux Terminal Shortcuts & Commands

- `Ctrl + C` — terminate the running process
- `Ctrl + Z` — suspend the current process (send it to background, stopped state)

- `fg` — bring a background/stopped job back to the foreground
- `bg` — resume a stopped job in the background

- `Ctrl + D` — exit the shell or signal end-of-input (EOF)
- `exit` — close the current shell/session

- `Ctrl + L` — clear the terminal screen
- `Ctrl + A` — move cursor to beginning of the line
- `Ctrl + E` — move cursor to end of the line
- `Ctrl + U` — delete from cursor to beginning of line
- `Ctrl + K` — delete from cursor to end of line
- `Ctrl + W` — delete the word before the cursor

- `Ctrl + R` — search command history (reverse incremental search)
- `Tab` — auto-complete commands, files, and directories

- `!!` — repeat the last command
