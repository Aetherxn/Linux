# System Information Commands

- `uname -a` — display kernel and system information
- `hostname` — display the system hostname
- `hostname -I` — display the system's IP address(es)
- `lscpu` — display CPU information
- `free -h` — display memory usage in human-readable format
- `df -h` — display disk space usage
- `du -sh dir` — display the size of a directory
- `lsblk` — list block devices (disks and partitions)
- `mount` — display mounted filesystems
- `uptime` — display system uptime and load average
- `who` — display logged-in users
- `whoami` — display the current user
- `id` — display the current user's UID, GID, and groups
- `ps aux` — display all running processes
- `top` — display real-time system and process information
